pyFiber

Python based percent crystallinity tool

pyFiber is an attempt at a user friendly program for determining the percent crystallinity of polymer and polymer nanocomposites. This node based approach is meant to mirror the user experience of parametric equilzers, and supports WAXD data (tested using Rigaku results) and LAMMPS outputs from the XRD package.
